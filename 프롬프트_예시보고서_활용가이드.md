## 1. 개요 및 구성

### 1-1. 왜 이 자료가 필요한가?

```
공인중개사의 경쟁력 = 정보력 + 분석력 + 전달력

AI 프롬프트 활용 시:
✓ 최신 시장 정보 수집 → 정보력 UP
✓ 체계적 분석 보고서 → 분석력 UP
✓ 전문적인 상담 자료 → 전달력 UP
```

### 1-2. 구성 파일 (4개 프롬프트 + 4개 예시보고서)

| 테마 | 프롬프트 파일 | 예시보고서 파일 | 핵심 주제 |
|------|--------------|----------------|----------|
| **테마1** | `01_하이엔드주거_분석_프롬프트.md` | `01_하이엔드주거_예시보고서.md` | 성수전략정비구역 재개발 |
| **테마2** | `02_브랜딩상권_분석_프롬프트.md` | `02_브랜딩상권_예시보고서.md` | 젠트리피케이션 방지 정책 |
| **테마3** | `03_IT지식산업_분석_프롬프트.md` | `03_IT지식산업_예시보고서.md` | IT진흥지구 기업유치 인센티브 |
| **테마4** | `04_팝업콘텐츠_분석_프롬프트.md` | `04_팝업콘텐츠_예시보고서.md` | 팝업스토어 / 용도변경 |

### 1-3. 프롬프트 vs 예시보고서 역할

| 구분 | 프롬프트 | 예시보고서 |
|------|---------|-----------|
| **용도** | AI에 입력하여 보고서 생성 | 완성된 결과물 참고용 |
| **버전** | 플레이스홀더 / 완성형 2가지 | 완성된 보고서 1개 |
| **사용 시점** | 최신 정보 필요할 때 | 빠른 참고 / 인쇄 시 |
| **장점** | 실시간 데이터 반영 | 즉시 활용 가능 |

## 2. 임장 전 활용법 (사전학습)

### 2-1. 목적

```
임장 전 사전학습 목표:
1. 해당 지역의 핵심 이슈 파악
2. 현장에서 확인할 포인트 정리
3. 고객 질문에 대비한 답변 준비
```

### 2-2. 활용 방법 3단계

#### Step 1: 예시보고서 읽기 (10~15분)

```
1. 해당 테마의 예시보고서 열기
2. 전체 구조 훑어보기 (목차 확인)
3. 핵심 데이터/수치 메모
4. Q&A 섹션 숙지 (고객 질문 대비)
```

**예시 (테마1: 하이엔드주거)**

```markdown
✓ 읽어야 할 파일: 01_하이엔드주거_예시보고서.md

✓ 메모할 핵심 수치:
- 총 세대수: 9,428세대
- 최고 층수: 65층 (250m)
- 1지구 지분 시세: 평당 1억~1.1억
- 예상 완공 시세: 84㎡ 기준 70~80억 (낙관)

✓ 숙지할 Q&A:
Q. 지금 매수해도 될까요?
A. "장기 투자(5~10년) 관점이라면 검토 가능.
   단, 최소 수십억 현금 필요, 갭투자 불가"
```

#### Step 2: AI로 최신 정보 업데이트 (선택, 5~10분)

임장 직전에 최신 정보가 필요하면 프롬프트 사용:

```
1. 프롬프트 파일의 "완성형 예시" 복사
2. ChatGPT / Claude / Gemini에 붙여넣기
3. 웹검색 활성화하여 실행
4. 최신 정보로 업데이트된 보고서 확인
```

**어떤 AI를 사용할까?**

| AI | 장점 | 단점 | 추천 상황 |
|----|------|------|----------|
| **ChatGPT** | 웹검색 강력, 대중적 | 유료 필요 | 최신 뉴스 검색 시 |
| **Claude** | 긴 문서 처리 우수 | 웹검색 제한적 | 분석/정리 시 |
| **Gemini** | 무료, 한글 양호 | 환각 주의 | 빠른 조회 시 |

#### Step 3: 체크리스트 인쇄 (5분)

```
1. 예시보고서의 "임장 시 확인사항" 섹션 찾기
2. 체크리스트 부분만 복사
3. A4 1장으로 인쇄
4. 현장에서 펜으로 체크
```

**인쇄용 체크리스트 예시 (테마1)**

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
■ 성수전략정비구역 임장 체크리스트
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

□ 1지구~4지구 경계 위치 파악
□ 현재 철거 진행 상황 확인
□ 한강 조망 가능 지점 확인
□ 성수역/뚝섬역 도보 거리 측정
□ 인근 부동산 방문 (시세 확인)

■ 부동산 인터뷰 질문
□ 현재 지분 거래 가격대?
□ 최근 실거래 사례?
□ 시공사 선정 예상 일정?
□ 분담금 예상액?
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 3. 임장 중 활용법 (현장 체크)

### 3-1. 스마트폰 활용

```
임장 중 스마트폰으로 할 일:
1. 체크리스트 사진 촬영 (펜으로 체크한 것)
2. 현장 사진/영상 촬영 (보고서용)
3. 부동산 소장님 인터뷰 녹음 (동의 받고)
4. 카카오맵으로 도보 거리 측정
```

### 3-2. 테마별 현장 확인 포인트

| 테마 | 반드시 확인할 것 | 사진 찍을 것 |
|------|-----------------|-------------|
| **테마1** 재개발 | 지구 경계, 철거 현황, 한강 조망 | 현장 전경, 한강뷰 지점 |
| **테마2** 상권 | 독립상점 비율, 공실 현황 | 거리 분위기, 간판들 |
| **테마3** IT지구 | 지식산업센터 외관, 입주기업 | 센터 로비, 안내판 |
| **테마4** 팝업 | 운영 중 팝업, 공실 공간 | 팝업 외관, 내부 |

### 3-3. 부동산 소장님 인터뷰 질문 (테마별)

#### 테마1: 하이엔드주거 (재개발)

```
1. "현재 지분 거래가 활발한가요? 최근 거래 사례는?"
2. "각 지구별 분위기는 어떤가요? 갈등 있나요?"
3. "시공사 선정 언제 될 것 같으세요?"
4. "분담금 대략 얼마로 예상하시나요?"
5. "투자 문의 많이 오나요? 어떤 분들이 주로 오세요?"
```

#### 테마2: 브랜딩상권 (입점 제한)

```
1. "프랜차이즈 입점 제한 때문에 불편한 점 있으세요?"
2. "입점 심사 절차가 까다롭나요?"
3. "임대료 인상이 억제되고 있다던데 실제로 그런가요?"
4. "공실이 많은 편인가요? 임차인 구하기 어려운가요?"
5. "어떤 업종이 들어오면 잘 될 것 같으세요?"
```

#### 테마3: IT지식산업 (기업유치)

```
1. "지식산업센터 입주율이 어떤가요? 공실 있나요?"
2. "어떤 기업들이 주로 입주해 있나요?"
3. "용적률 인센티브 받은 건물이 있나요?"
4. "분양가/임대료가 최근에 어떻게 변했나요?"
5. "스타트업이 많이 찾나요, 대기업이 많이 찾나요?"
```

#### 테마4: 팝업콘텐츠 (용도변경)

```
1. "팝업 공간 임대료가 어느 정도인가요?"
2. "얼마 전에 예약해야 공간 잡을 수 있나요?"
3. "용도변경 했나요? 절차가 복잡했나요?"
4. "명품 브랜드 팝업도 들어오나요?"
5. "최근 팝업 트렌드가 어떻게 변하고 있나요?"
```

## 4. 임장 후 활용법 (보고서 작성)

### 4-1. 목적

```
임장 후 보고서 작성 목표:
1. 현장 확인 내용 정리 및 기록
2. 고객 상담용 자료 제작
3. 나만의 지역 분석 자산 축적
```

### 4-2. AI로 맞춤 보고서 생성 (15~20분)

#### Step 1: 프롬프트 템플릿 복사

해당 테마의 프롬프트 파일에서 **"플레이스홀더 버전"** 복사

#### Step 2: 현장 정보 입력

`[...]` 부분을 임장에서 수집한 정보로 교체

**입력 예시 (테마1 재개발)**

```markdown
## 입력 데이터

| 항목 | 입력값 |
|------|--------|
| 분석 지역 | 성수전략정비구역 1지구 |
| 분석 기준일 | 2026-02-22 (임장일) |
| 분석 목적 | 고객 A씨 투자 상담용 |

## 현장 수집 정보

- 1지구 경계: ○○부동산 앞에서 시작
- 철거 현황: 약 30% 철거 진행
- 한강 조망: ○○빌라 옥상에서 확인 (15층 이상 조망 가능 예상)
- 부동산 소장님 의견: "시공사 선정 2026년 하반기 예상"
- 지분 시세: 소형 평당 1.1억, 대형 평당 8,500만원
```

#### Step 3: AI에 입력하여 보고서 생성

```
1. ChatGPT / Claude / Gemini에 수정한 프롬프트 붙여넣기
2. "위 정보를 바탕으로 분석 보고서를 작성해주세요" 입력
3. 생성된 보고서 검토
4. 필요시 수정 요청 ("○○ 부분 더 자세히 써줘")
```

#### Step 4: 보고서 저장 및 활용

```
1. 생성된 보고서를 워드/한글/노션에 저장
2. 파일명: "2026-02-22_성수1지구_A고객용.md"
3. 고객 상담 시 출력하여 제공
4. 팀 내 공유 (사무실 자료실에 저장)
```

### 4-3. 보고서 활용 팁

| 상황 | 활용 방법 |
|------|----------|
| **고객 방문 상담** | 보고서 출력하여 함께 보며 설명 |
| **전화 상담** | 보고서 보면서 핵심 수치 안내 |
| **카카오톡 상담** | 보고서 PDF 변환하여 전송 |
| **블로그/SNS** | 보고서 일부 발췌하여 콘텐츠 제작 |
| **사무실 자료** | 지역별 폴더에 정리하여 팀 공유 |

## 5. 고객 상담 시 활용법

### 5-1. 고객 유형별 활용 전략

#### 투자자 고객 (재개발 지분)

```
■ 사용할 자료: 테마1 하이엔드주거 보고서

■ 상담 포인트:
1. 지구별 진행 상황 비교표 보여주기
2. 예상 시세 (조합원가 vs 완공 후) 설명
3. 리스크 체크리스트 함께 확인
4. SWOT 분석 표로 객관적 안내

■ 대화 예시:
"이 표 보시면요, 1지구가 가장 핵심이고 4지구가 가장 빠릅니다.
현재 평당 1억 지분이 완공 후 1.5억 이상 될 수 있는데요,
대신 이 5가지 리스크는 꼭 알고 계셔야 합니다."
```

#### 상가 임차인 고객 (창업 희망)

```
■ 사용할 자료: 테마2 브랜딩상권 보고서

■ 상담 포인트:
1. 입점 제한 업종 리스트 확인
2. 심사 절차 및 기간 안내
3. 예외 승인 가능성 설명
4. 임대료 시세 정보 제공

■ 대화 예시:
"성수동 아뜰리에길은 프랜차이즈 입점이 제한됩니다.
하지만 독립 브랜드시라면 오히려 좋은 환경이에요.
입점 심사는 보통 2~4주 걸리고, 이 서류들이 필요합니다."
```

#### 기업 고객 (사무실 이전)

```
■ 사용할 자료: 테마3 IT지식산업 보고서

■ 상담 포인트:
1. IT진흥지구 인센티브 설명 (용적률, 세제)
2. 권장 업종 해당 여부 확인
3. 지식산업센터 vs 일반 오피스 비교
4. 주요 입주 기업 소개 (무신사, 하고 등)

■ 대화 예시:
"IT 업종이시면 성수가 최적입니다.
취득세가 35~50% 감면되고요, 용적률도 560%까지 가능해요.
무신사, 29cm 같은 기업들이 다 이 지역에 있습니다."
```

#### 브랜드 마케팅 담당자 (팝업 희망)

```
■ 사용할 자료: 테마4 팝업콘텐츠 보고서

■ 상담 포인트:
1. 팝업 공간 임대료 시세 안내
2. 예약 리드타임 (1~3개월 전)
3. 계약 조건 및 특약 사항
4. 성공 팝업 사례 소개

■ 대화 예시:
"연무장길 50평 기준 일 200~300만원 정도입니다.
인기 공간은 2~3개월 전에 예약해야 하고요,
최근에 ○○ 브랜드가 여기서 팝업 해서 대박 났어요."
```

### 5-2. Q&A 대응 스크립트

예시보고서의 "고객 응대 Q&A" 섹션을 활용하여 자연스럽게 대응:

**테마1 예시**

| 고객 질문 | 권장 답변 |
|----------|----------|

## 9. 자주 묻는 질문 (FAQ)

### Q1. AI 없이도 활용할 수 있나요?

```
A. 네, 가능합니다.

예시보고서만 활용하는 방법:
1. 예시보고서를 읽고 핵심 내용 숙지
2. 체크리스트 인쇄하여 현장 사용
3. 보고서 양식 참고하여 직접 작성

AI 사용 시 장점:
- 최신 정보 반영 가능
- 맞춤 보고서 빠르게 생성
- 반복 업무 시간 절약
```

### Q2. 어떤 AI가 가장 좋나요?

```
A. 상황에 따라 다릅니다.

| 상황 | 추천 AI |
|------|---------|
| 최신 뉴스/시세 검색 | ChatGPT (웹검색 ON) |
| 긴 보고서 생성 | Claude |
| 무료로 빠르게 | Gemini |
| 한글 품질 중시 | Claude or ChatGPT |

Tip: 중요한 보고서는 2개 AI에 넣어서 교차 검증 추천
```

### Q3. 프롬프트 수정해도 되나요?

```
A. 물론입니다. 적극 권장합니다.

수정 가능 부분:
- 입력 데이터 (고객 조건, 지역 등)
- 출력 형식 (표 추가, 섹션 변경)
- 분석 항목 (가중치 조정)

수정 예시:
"## 입력 데이터"에 고객 예산, 투자 목적 추가
"## 추가 요청: 경쟁 단지 3개와 비교해주세요"
```

```
A. AI 환각(Hallucination) 주의 필요합니다.

검증 필수 항목:
□ 가격/시세 (호갱노노, KB부동산에서 확인)
□ 날짜/일정 (공식 발표 확인)
□ 법규/조례 (국가법령정보센터 확인)

Tip: "출처도 함께 알려줘" 프롬프트에 추가하면
AI가 참고한 출처를 표시해줍니다.
```
