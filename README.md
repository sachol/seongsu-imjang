> **대상**: 초보/개업 공인중개사

## 개요

이 폴더는 RSA 총동문회 성수동 임장 수업을 위한 **AI 프롬프트 템플릿**과 **예시 보고서**를 포함합니다.

ChatAPT_Week4_Prompts의 구조를 참고하여, 성수동의 4가지 핵심 테마에 맞춤화된 프롬프트를 제공합니다.

## 사용 방법

### 방법 1: 임장 전 사전학습

1. **예시 보고서** 파일을 읽고 각 테마의 핵심 내용을 파악합니다.
2. 임장 전 성수동의 4가지 테마에 대한 배경 지식을 습득합니다.

### 방법 2: AI로 최신 정보 조회

1. **프롬프트 템플릿** 파일의 "완성형 예시"를 복사합니다.
2. ChatGPT, Claude, Gemini 등에 붙여넣기합니다.
3. AI가 웹 검색을 통해 최신 정보를 수집하여 보고서를 생성합니다.

### 방법 3: 임장 후 보고서 작성

1. **프롬프트 템플릿**의 "플레이스홀더 버전"을 복사합니다.
2. 임장에서 확인한 내용을 입력 데이터에 채웁니다.
3. AI에 입력하여 맞춤형 보고서를 생성합니다.

## 테마별 핵심 요약

### 테마1: 하이엔드 주거 (성수전략정비구역)

```
핵심: 한강변 65층 초고층 약 1만 세대 재개발
- 1지구: 3,019세대, 평당 1억+ 지분
- 2지구: 2,609세대, 입찰 유찰
- 3지구: 설계 변경 중
- 4지구: 1,439세대, 가장 빠른 진행
완공 후 예상 시세: 84㎡ 기준 70~80억 (낙관)
```

### 테마2: 브랜딩 상권 (입점 제한 구역)

```
핵심: 프랜차이즈 입점 제한으로 독립 상권 보존
- 제한 업종: 휴게음식점, 일반음식점, 제과점, 화장품판매업
- 적용 지역: 서울숲길, 아뜰리에길 (성수1가2동)
- 예외: 상호협력주민협의체 심의 후 승인 시
- 효과: 임대료 인상률 2.72% (법정 5% 이하)
```

### 테마3: IT지식산업 (기업유치)

```
핵심: IT/디자인/미디어 기업 인센티브 제공
- 용적률 완화: 400% → 560% (1.4배)
- 취득세 감면: 35~50%
- 지식산업센터: 66개 (4배 확대)
- 대표 기업: 무신사, 하고, 29cm
```

### 테마4: 팝업 & 콘텐츠 (용도변경)

```
핵심: 대한민국 팝업스토어 성지
- 2025년 서울 팝업 비중 1위
- 임대료: 50평 기준 월 1,500~3,000만원
- 준공업지역: 건폐율 60%, 용적률 400%
- 용도변경: 창고→근린생활시설 시 1~3개월 소요
```

## 임장 보고서 질문 (테마별 3개)

### 테마1: 하이엔드 주거
1. 한강 조망과 서울숲 접근성이 어떻게 달라지나요? 트리마제 대비 경쟁력은?
2. 현재 지분 거래가격과 예상 분양가의 갭은 어느 정도인가요?
3. 각 지구별 사업 지연 가능성과 분담금 변동 리스크는?

### 테마2: 브랜딩 상권
1. 젠트리피케이션 방지 정책이 실제로 어떤 변화를 가져왔나요?
2. 프랜차이즈 입점 제한이 임대료와 공실률에 어떤 영향을 주고 있나요?
3. 입점 심사를 받아본 경험이 있나요? 절차가 까다롭나요?

### 테마3: IT지식산업
1. 어떤 종류의 기업들이 주로 입주해 있나요? 산업 간 시너지가 있나요?
2. 용적률 완화와 세제 감면을 실제로 적용받은 사례가 있나요?
3. 지식산업센터 분양가와 임대료 추세는 어떤가요?

### 테마4: 팝업 & 콘텐츠
1. 공장/창고 건물이 팝업 공간으로 어떻게 변신했나요? 용도변경 절차는?
2. 팝업 공간 임대료와 계약 조건은 어떻게 되나요?
3. 최근 팝업 트렌드는 어떻게 변하고 있나요? 명품 브랜드 진출 영향은?
