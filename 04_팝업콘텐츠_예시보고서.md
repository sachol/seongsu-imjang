## 1. 팝업스토어 시장 현황

### 1-1. 왜 '팝업스토어'인가?

```
팝업스토어 (Pop-up Store)
= 짧은 기간 동안 운영되는 임시 매장
= 브랜드 체험, 신제품 홍보, MZ세대 마케팅 핵심 채널

성장 배경:
- 온라인 브랜드의 오프라인 체험 필요성 증가
- SNS 바이럴 마케팅 효과
- MZ세대 "경험 소비" 트렌드
- 짧은 기간, 높은 임팩트
```

### 1-2. 성수동 = 대한민국 팝업 성지

| 항목 | 성수동 |
|------|--------|
| **서울 팝업 비중** | 2025년 기준 **1위** |
| **월 평균 팝업 수** | 50개+ (추정) |
| **핵심 거리** | 연무장길, 서울숲길 |
| **대표 미디어** | 성수동고릴라 (팝업 전문) |

### 1-3. 성수동이 팝업 성지가 된 5가지 이유

| 순위 | 이유 | 설명 |
|------|------|------|
| 1 | **MZ세대 집객력** | 20~30대 트렌드 소비층 밀집 |
| 2 | **SNS 바이럴** | "성수 갔다 옴" = 인스타그래머블 |
| 3 | **공간 독특함** | 공장/창고 리모델링 → 힙한 공간감 |
| 4 | **접근성** | 2호선 성수역 (강남 10분) |
| 5 | **문화 집적** | 카페, 갤러리, 편집샵 밀집 |

## 2. 연무장길 상권 분석

### 2-1. 상권 개요

```
■ 위치
성수역 3번 출구 → 연무장길 방향
(대략 성수역~뚝섬역 사이)

■ 특징
- 길이: 약 500m
- 분위기: 공장 리모델링 상가 밀집
- 별명: "팝업 성지", "K-힙스터 거리"

■ 주요 골목
- 연무장길 메인 도로
- 연무장5길 (카페 골목)
- 서울숲2길 (아뜰리에길)
```

### 2-2. 유동인구 특성

| 항목 | 내용 |
|------|------|
| **주요 연령** | 20~35세 (MZ세대) |
| **성비** | 여성 약간 우세 (6:4) |
| **방문 목적** | 팝업 방문, 카페, 데이트, SNS 촬영 |
| **피크 타임** | 주말 오후 2~6시 |
| **주중 vs 주말** | 주말 유동인구 3~5배 |

### 2-3. 팝업 공간 유형

| 유형 | 면적 | 특징 | 대표 사례 |
|------|------|------|----------|
| **독립 건물 (통임대)** | 100~300평 | 브랜드 단독 사용, 대형 팝업 | 명품 브랜드 |
| **복합 공간** | 50~100평 | 여러 브랜드 공유 | 쇼룸, 전시 |
| **소형 공간** | 20~50평 | 스타트업, 소규모 브랜드 | 신진 브랜드 |
| **야외/루프탑** | 다양 | 계절 한정, 이벤트 | 시즌 팝업 |

## 3. 팝업 공간 임대료 시세 (2026년 기준 추정)

### 3-1. 유형별 임대료

| 유형 | 면적 | 일일 임대료 | 주간(7일) | 월 임대료 |
|------|------|-------------|-----------|-----------|
| **프라임 위치 (대로변)** | 100평 | 300~500만원 | 1,500~2,500만원 | 4,000~8,000만원 |
| **일반 위치** | 50평 | 100~200만원 | 500~1,000만원 | 1,500~3,000만원 |
| **소형 공간** | 20평 | 50~100만원 | 250~500만원 | 700~1,500만원 |

### 3-2. 가격 결정 요인

```
■ 가격 상승 요인
+ 연무장길 메인 도로변
+ 대형 유리창 (노출도 높음)
+ 기존 인테리어 완비
+ 주말/시즌 (성수기)
+ 명품 브랜드 이전 입점 이력

■ 가격 하락 요인
- 골목 안쪽 위치
- 지하/2층 이상
- 인테리어 미비 (공사 필요)
- 평일/비수기
```

### 3-3. 비용 구조 (예시: 50평, 2주 팝업)

| 항목 | 금액 | 비고 |
|------|------|------|
| 공간 임대료 | 800만원 | 50평 × 2주 |
| 보증금 | 400만원 | 임대료 50% (반환) |
| 인테리어 | 500~1,000만원 | 컨셉에 따라 |
| 유틸리티 | 50만원 | 전기/수도/인터넷 |
| 원상복구 | 100~300만원 | 계약 조건 |
| **총합** | **약 2,000~2,500만원** | 2주 기준 |

## 4. 준공업지역 용도변경 가이드

### 4-1. 성수동 준공업지역 기본 규정

| 항목 | 기준 |
|------|------|
| **건폐율** | 60% |
| **용적률** | 400% (기본) / 560% (IT기업 입주 시) |
| **허용 용도** | 공장, 창고, 지식산업센터, 상업시설 일부 |

### 4-2. 팝업 공간 운영 시 용도 확인

```
■ 용도변경 필요 여부

기존 용도 → 팝업 용도 → 필요 여부
────────────────────────────────────
근린생활시설 → 팝업스토어 → 대체로 불필요
창고/공장 → 팝업스토어 → 용도변경 필요
사무실 → 팝업스토어 → 일부 가능/확인 필요

⚠️ 주의: 정확한 용도는 건축물대장 확인 필수
```

### 4-3. 용도변경 절차 (창고 → 근린생활시설)

```
[1] 건축물대장 확인
    └─ 현재 용도 확인 (정부24, 건축행정시스템)
           │
           ▼
[2] 용도변경 가능 여부 검토
    └─ 건축사/구청 상담
    └─ 준공업지역 허용 용도 확인
           │
           ▼
[3] 설계 및 서류 준비
    └─ 건축사 도면 작성
    └─ 소방, 위생 등 기준 충족
           │
           ▼
[4] 구청 신고/허가
    └─ 신고 (100㎡ 미만, 단순 변경)
    └─ 허가 (100㎡ 이상, 복잡한 변경)
           │
           ▼
[5] 공사 및 준공
    └─ 소방/위생 시설 설치
    └─ 사용승인
           │
           ▼
[6] 건축물대장 변경
    └─ 용도 변경 완료
```

### 4-4. 용도변경 비용/기간 (대략)

| 항목 | 비용 | 기간 |
|------|------|------|
| 건축사 비용 | 100~300만원 | - |
| 소방 시설 | 300~500만원 | - |
| 인허가 수수료 | 10~50만원 | - |
| **총 기간** | - | **1~3개월** |

### 4-5. 용도변경 없이 운영하는 방법

```
■ 단기 이벤트 활용
- 일시적 사용 (이벤트 신고)
- 기간 제한적으로 용도변경 없이 가능

■ 기존 용도 활용
- 근린생활시설로 이미 등록된 공간 임대
- "전시장", "소매점" 용도 공간 활용

■ 주의사항
- 장기/상시 운영 시 단속 위험
- 화재/안전 사고 시 보험 적용 불가 가능
- 건물주와 책임 소재 명확히
```

## 5. 팝업 공간 임대 계약 가이드

### 5-1. 계약 유형

| 유형 | 기간 | 특징 | 임대료 |
|------|------|------|--------|
| **단기** | 1~7일 | 이벤트/론칭 | 일 단가 |
| **중기** | 2~4주 | 일반 팝업 | 주/일 단가 |
| **장기** | 1~6개월 | 쇼룸/시즌 | 월 단가 (할인) |

### 5-2. 임대인(건물주) 체크리스트

```
■ 계약 전
[ ] 건축물대장 용도 확인 (팝업 운영 가능 용도인지)
[ ] 소방/안전 시설 점검
[ ] 보험 가입 여부 확인

■ 계약 시
[ ] 원상복구 범위 및 비용 명확화
[ ] 보증금/월세 지급 조건
[ ] 촬영/소음/주류 등 제한 사항
[ ] 제3자 양도 금지

■ 운영 중
[ ] 정기 점검 (안전, 시설)
[ ] 민원 대응 (주민, 상가)
```

### 5-3. 임차인(브랜드) 체크리스트

```
■ 계약 전
[ ] 공간 용도 확인 (용도변경 필요 여부)
[ ] 인테리어 공사 가능 범위
[ ] 전기/수도/인터넷 용량 확인
[ ] 주차/물류 동선 확인

■ 계약 시
[ ] 원상복구 범위 정확히 (사진 촬영)
[ ] 보험 가입 의무 여부
[ ] 조기 철수 시 위약금
[ ] 연장 옵션

■ 운영 후
[ ] 원상복구 완료 확인
[ ] 보증금 반환 일정
```

### 5-4. 계약서 특약 예시

```markdown
■ 특약 예시 1 (원상복구)

"임차인은 임대 기간 종료 시 별첨 사진에 명시된 상태로 원상복구한다.
단, 임대인이 서면 동의한 시설물은 원상복구 대상에서 제외한다."

■ 특약 예시 2 (용도변경)

"본 공간은 '근린생활시설(소매점)' 용도로 등록되어 있으며,
임차인은 해당 용도 범위 내에서 팝업스토어를 운영한다.
용도 외 사용으로 인한 모든 책임은 임차인에게 있다."

■ 특약 예시 3 (소음/민원)

"임차인은 22시 이후 음악/행사 소음을 자제하며,
주민 민원 발생 시 임대인의 지시에 따라 조치한다."

■ 특약 예시 4 (보험)

"임차인은 운영 기간 동안 영업배상책임보험에 가입하고,
보험증권 사본을 임대인에게 제출한다."
```

## 6. 주요 팝업 사례 (2025~2026년)

### 6-1. 최근 성공 사례

| 브랜드 | 기간 | 위치 | 컨셉 | 성과 |
|--------|------|------|------|------|
| **디올** | 2주 | 연무장길 | 럭셔리 쇼룸 | SNS 바이럴, 대기줄 |
| **넷플릭스** | 1주 | 서울숲2길 | 드라마 체험관 | 2만명+ 방문 |
| **뉴발란스** | 3주 | 연무장길 | 스니커즈 팝업 | MZ 타깃 성공 |
| **스타벅스** | 시즌 | 성수역 인근 | 한정판 굿즈 | 오픈런 화제 |

### 6-2. 성공 요인 분석

```
■ 컨셉 (Concept)
- SNS 공유하고 싶은 포토존
- 한정판/체험 요소
- 브랜드 세계관 몰입

■ 위치 (Location)
- 연무장길 메인 도로
- 유동인구 동선 상
- 대형 유리창 노출

■ 바이럴 (Viral)
- 인플루언서 초청
- 인스타그래머블 공간
- 해시태그 이벤트

■ 타이밍 (Timing)
- 신제품 출시와 연계
- 주말/공휴일 집중
- 시즌 이슈 활용
```

### 6-3. 명품 브랜드 진출 트렌드

```
최근 동향:
- 루이비통, 디올, 구찌 등 명품 브랜드 성수 진출
- 연무장길에 쇼룸/팝업 개설
- 젊은층 접점 확대 목적

영향:
- 임대료 상승 압력
- 상권 고급화 가속
- 기존 독립 브랜드와 경쟁
```

## 7. 고객 응대 Q&A

### Q1. 팝업 공간 임대료가 얼마인가요?

**A.**
> "위치와 면적에 따라 다릅니다.
>
> **대략적인 시세 (연무장길 기준)**:
> | 면적 | 일 단가 | 주 단가 | 월 단가 |
> |------|---------|---------|---------|
> | 50평 | 100~200만원 | 500~1,000만원 | 1,500~3,000만원 |
> | 100평 | 300~500만원 | 1,500~2,500만원 | 4,000~8,000만원 |
>
> ※ 성수기(연말, 봄/가을), 프라임 위치는 +30~50%"

---

### Q2. 얼마 전에 예약해야 하나요?

**A.**
> "좋은 위치는 **2~3개월 전** 예약 권장합니다.
>
> **예약 시점**:
> - 프라임 위치: 2~3개월 전
> - 일반 위치: 1~2개월 전
> - 비수기: 2~4주 전도 가능
>
> **성수기 (예약 경쟁 심함)**:
> - 3~5월 (봄 시즌)
> - 9~11월 (가을 시즌)
> - 12월 (연말 시즌)"

---

### Q3. 용도변경 없이 팝업 운영이 가능한가요?

**A.**
> "공간의 기존 용도에 따라 다릅니다.
>
> **용도변경 불필요**:
> - 근린생활시설(소매점, 전시장) 등록 공간
> - 단기 이벤트 형태 운영
>
> **용도변경 필요**:
> - 창고, 공장 등 비상업 용도
> - 장기/상시 운영 계획 시
>
> **확인 방법**:
> - 건축물대장 열람 (정부24)
> - 구청 건축과 문의"

---

### Q4. 건물주인데 팝업 공간으로 임대하려면?

**A.**
> "다음 사항을 확인하세요:
>
> **준비사항**:
> 1. 건축물대장 용도 확인
> 2. 소방/안전 시설 점검
> 3. 건물 보험 확인 (영업배상책임)
>
> **임대 채널**:
> - 팝업 전문 플랫폼 (데이포유, 팝가 등)
> - 공간 임대 플랫폼 (스페이스클라우드 등)
> - 부동산 중개
>
> **주의사항**:
> - 원상복구 조건 명확히
> - 소음/민원 대응 방안
> - 보험 가입 요구"

## 8. 임장 시 확인사항 체크리스트

### 현장 확인 (Physical Check)

```
■ 연무장길 탐방
[ ] 성수역 3번 출구 → 연무장길 도보 이동
[ ] 현재 운영 중인 팝업스토어 3곳 이상 방문
[ ] 공실/임대 안내 공간 확인
[ ] 명품 쇼룸 위치 확인
```
